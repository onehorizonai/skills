# Security Policy

## Reporting a Vulnerability

Please report suspected security issues privately by emailing security@onehorizon.ai.

Do not open a public issue for vulnerabilities. Include the affected version, impacted files or configuration, reproduction steps, and any relevant proof of concept details.

We aim to acknowledge reports within 3 business days and provide status updates while the issue is being reviewed.

## Supported Versions

Security fixes are handled on the main branch and released through the current published plugin version.
